# 肿瘤标签染色与可视化

> 分支: `bridge-optimization` | 日期: 2026-06-26

---

## 一、背景

MAISI 的 132 类 label 使用整数 ID 标识不同的解剖结构和病灶。在默认灰度显示下，不同标签之间几乎无法区分——肝(1)和肝肿瘤(26)都是灰度值，肉眼看不出区别。染色（colorize）就是为每个标签 ID 分配一个 RGB 颜色，让不同结构在可视化时一目了然。

这对验证桥接管线的输出尤其重要——肿瘤是否画在了正确的器官内、肿瘤区域与周围器官的边界是否合理，只有在彩色显示下才能快速判断。

---

## 二、相关文件

| 文件 | 功能 |
|------|------|
| `Relate/label_colorize.py` | 核心：将 132 类 label NIfTI 上色为 RGB 图像，输出彩色 NIfTI + PNG 截图 + 颜色对照表 |
| `Relate/label_colormap_export.py` | 导出 3D Slicer (.ctbl) 和 ITK-SNAP (.txt) 颜色映射文件，让外部查看器自动显示彩色 |
| `Relate/bridge_maisi_mask.py` | 桥接管线：在 mask 上画肿瘤，使用 `ORGAN_LABEL_MAP` 和 `TUMOR_LABEL_MAP` 定义器官→肿瘤标签映射 |

## 三、颜色映射设计

### 3.1 设计原则

1. **同类器官用相近色系** — 5个肺叶都是蓝色系的5种深浅，12对肋骨左偏黄右偏粉
2. **肿瘤用醒目红色系** — 与正常器官形成强对比，一眼就能定位肿瘤
3. **预留扩展空间** — 第四步新增的12种肿瘤标签(133-144)各有独立的红色深浅
4. **背景纯黑，body envelope 深灰** — 不干扰正常结构的辨识

## 四、器官→肿瘤标签映射

桥接管线 (`bridge_maisi_mask.py`) 中定义了两个映射表：

### ORGAN_LABEL_MAP（器官名→器官标签ID）

```python
"liver":     1,    "pancreas":  4,    "kidney":    5,
"colon":     62,   "esophagus": 11,   "lung":      20,
"bone":      21,   # 原有7个
"stomach":   12,   "bladder":   15,   "prostate":  118,
# ... 新增12个
```

### TUMOR_LABEL_MAP（器官名→肿瘤标签ID）

```python
"liver":     26,   "pancreas":  24,   "kidney":    116,
"colon":     27,   "esophagus": 133,  "lung":      23,
"bone":      128,  # 原有7个
"stomach":   134,  "bladder":   135,  "prostate":  136,
# ... 新增12个
```

桥接管线在指定器官上画肿瘤时，用 `TUMOR_LABEL_MAP` 获取对应的肿瘤标签ID，将该区域的器官标签替换为肿瘤标签。ControlNet 读取这个合并后的 label，就会在肿瘤区域生成肿瘤纹理。

## 五、使用方式

### 5.1 单文件染色

```powershell
python Relate\label_colorize.py --label output\merged_label.nii.gz
```

输出：
- `merged_label_colored.nii.gz` — RGB NIfTI（每个voxel是3通道uint8颜色）
- `merged_label_colored_axial.png` — 轴位截图（自动定位到肿瘤中心切片）
- `merged_label_colored_coronal.png` — 冠状位截图
- `merged_label_legend.png` — 颜色对照表

### 5.2 批量染色

```powershell
python Relate\label_colorize.py --scan-dir output\
```

自动扫描 `output/` 下所有任务目录中的 `04_final_merged/merged_label.nii.gz`。

### 5.3 只生成PNG（不生成NIfTI）

```powershell
python Relate\label_colorize.py --label output\merged_label.nii.gz --png-only
```

### 5.4 导出3D Slicer / ITK-SNAP 颜色映射

```powershell
python Relate\label_colormap_export.py --label output\merged_label.nii.gz
```

输出 `.ctbl`（3D Slicer）和 `.txt`（ITK-SNAP）格式，在外部查看器中打开 label 时自动显示彩色。

## 七、技术细节

### 7.1 柿子标签的处理

MAISI 原始 label 中，116（左肾囊肿）和 117（右肾囊肿）在步骤1.5被清除（还原为器官标签5/14），桥接管线画的新肿瘤使用标签26（肝肿瘤）等。染色时这三者都是红色系但深浅不同，不会混淆。

### 7.2 PNG切片选择

`save_png_slice` 优先查找标签26（肝肿瘤）的位置，取其中心切片。如果文件中没有肿瘤标签，则取中间切片。这个逻辑对桥接管线的输出特别有用——自动展示肿瘤最明显的切面。

### 7.3 未映射标签的回退

不在 `LABEL_COLORS` 映射表中的标签ID，使用 `_default_color()` 自动生成灰度色（`60 + (label_id * 3) % 180`），保证任何标签都有颜色，不会遗漏。

### 7.4 RGB NIfTI格式

nibabel 保存 RGB NIfTI 需要形状为 (W, H, D, 3)，而 numpy 数组是 (D, H, W, 3)，所以保存前做了 `transpose(2,1,0,3)` 转换，并设置 `header.set_intent('vector')` 标识为向量图像。

## 八、ITK-SNAP 中打开染色 CT 的流程

### 8.1 软件介绍

ITK-SNAP 是一款开源的医学图像分割与可视化软件，支持 NIfTI 格式，可以同时加载 CT 图像和 label，以彩色叠加方式显示分割结果。下载地址：http://www.itksnap.org/

### 8.2 打开 CT + 染色 Label 的步骤

**第一步：导出颜色映射文件**

```powershell
python Relate\label_colormap_export.py --label output\merged_label.nii.gz
```

生成 `_colormap.ctbl` 和 `_itksnap_labels.txt` 两个文件。

**第二步：打开 ITK-SNAP，加载图像**

1. 菜单 → `File` → `Open Main Image`
2. 选择 CT 文件（如 `merged_label_*_image.nii.gz`）
3. 弹出向导，确认图像参数（spacing、origin），点 `Next` → `Finish`

**第三步：加载 Label 叠加层**

1. 菜单 → `File` → `Open Segmentation`（或工具栏的 "Open Segmentation" 按钮）
2. 选择 label 文件（如 `merged_label.nii.gz`）
3. ITK-SNAP 会以随机颜色显示 label——此时颜色与我们的定义不一致，需要加载颜色映射

**第四步：加载自定义颜色映射**

1. 菜单 → `Tools` → `Label Editor`
2. 在 Label Editor 窗口中，点击 `Load` 按钮（底部）
3. 选择之前导出的 `_itksnap_labels.txt` 文件
4. 颜色会立即更新为我们的自定义配色：肝脏红褐、肿瘤纯红、肺叶蓝色系等

**第五步：浏览和检查**

1. 使用鼠标滚轮或滑动条在切片间移动
2. 工具栏的十字线工具可以同时在三个正交平面定位
3. 右侧 Segmentation 面板可以勾选/取消特定标签的显示
4. 支持三维体渲染：菜单 → `Tools` → `Volume Rendering`

### 8.3 方式二：直接打开 RGB NIfTI（更简单）

如果不需要交互式操作，可以直接打开染色后的 RGB NIfTI：

1. 菜单 → `File` → `Open Main Image`
2. 选择 `merged_label_colored.nii.gz`（由 `label_colorize.py` 生成）
3. 图像直接以彩色显示，无需额外配置颜色映射

缺点：无法单独控制各标签的显示/隐藏，因为这不是一个分割文件，而是一张 RGB 图像。

### 8.4 常用操作速查

| 操作 | 方法 |
|------|------|
| 切换显示模式（分割叠加/轮廓/透明） | 工具栏 `Segmentation Rendering` 下拉菜单 |
| 隐藏某个标签 | Label Editor 中取消勾选该标签 |
| 只显示肿瘤 | Label Editor 中只勾选肿瘤标签（23, 24, 26, 27, 128, 133-144） |
| 调整叠加透明度 | 菜单 → `Settings` → `Display Layout` → `Overlay Opacity` |
| 测量肿瘤体积 | Label Editor 中选中肿瘤标签，右侧显示 Volume (mm³) |
| 截图保存 | 菜单 → `File` → `Save Screenshot` |
| 同时加载多组对比 | 打开第一组后，`File` → `Open Additional Image` 加载另一组 CT |
