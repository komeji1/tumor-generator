## 列说明

| 列 | 说明 |
|----|------|
| # | 序号 |
| organ | 目标器官 |
| size | 尺寸档 (small: 5-10mm, medium: 10-20mm, large: 20-35mm) |
| r(mm) | 实际采样的肿瘤半径 |
| voxels | 肿瘤体素数 |
| overlap | 肿瘤在器官内的比例 |
| 约束 | Y=安全约束触发(裁剪到器官内), -=未触发(保留浸润) |
| center(i,j,k) | 肿瘤重心的 NIfTI 体素坐标 (0-based) |
| world(x,y,z)mm | 肿瘤重心的 NIfTI 世界坐标 (mm, 经 affine 变换) |
| bbox(i,j,k) | 肿瘤的体素 bounding box 范围 |

## 使用方法

```bash
# 将任意示例文件传给 MAISI 生成含肿瘤的 CT
python scripts/infer_image_from_mask.py \
  --mask Relate/example_output/sample_*_liver_large_tumor_v43.nii.gz \
  --environment configs/environment_rflow-ct.json \
  --inference configs/config_infer_16g_256x256x128.json \
  --config configs/config_network_rflow.json
```
